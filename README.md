# 釧路コミュニティチャット

北海道釧路市の地域住民向けコミュニティチャットWebサイトです。

## ファイル構成

```
kushiro-chat/
├── index.html          # ログイン・エントリーページ
├── chat.html           # メインチャットページ
├── css/
│   ├── style.css       # グローバルスタイル
│   └── chat.css        # チャット画面専用スタイル
├── js/
│   ├── storage.js      # データ管理・CSV処理
│   ├── main.js         # ログインページ処理
│   └── chat.js         # チャット機能メイン
├── csv/
│   ├── users.csv       # ユーザーデータ（テンプレート）
│   ├── rooms.csv       # 掲示板データ（テンプレート）
│   └── messages.csv    # メッセージデータ（テンプレート）
└── images/
    ├── logo.svg        # サイトロゴ
    └── bg-night.svg    # 背景画像（夜景）
```

## 機能

- **ニックネーム設定** - アバターカラーも選択可能
- **複数掲示板** - カテゴリ・タグ分類あり
- **新規掲示板作成** - ユーザーが自由に作成
- **画像・動画添付** - メッセージに添付可能
- **タグ付け** - 各メッセージにタグを追加
- **CSVエクスポート** - データをCSVで保存・バックアップ
- **擬似リアルタイム** - 3秒間隔でポーリング更新

## データ保存について

データはブラウザのlocalStorageに保存されます。
CSVエクスポート機能を使うことで以下のファイルを保存できます：
- `kushiro_users.csv` - ユーザー情報
- `kushiro_rooms.csv` - 掲示板情報  
- `kushiro_messages.csv` - メッセージ一覧

## 開始方法

ローカルサーバーで `index.html` を開いてください。
（ファイルの直接開放ではなく、HTTPサーバー経由を推奨）

例：
```
python3 -m http.server 8080
```

## デザインテーマ

釧路の夜景・港の灯り・霧をイメージした暗いカラーリング：
- 背景：深夜の海色（#060c18, #0a0e1a）
- アクセント：港の灯り（アンバー #f4a620）
- サブカラー：霧に滲む水面（シアン #4dd0e1）
